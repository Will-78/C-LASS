High-level Requirements

Web UI

Provide a web UI where students ask natural-language questions about static-analysis CTFs and receive grounded, consistent, instructional answers.

Retrieval-Augmented Generation

Implement Retrieval-Augmented Generation that combines an LLM with a curated corpus of static-analysis CTF write-ups, docs, and explanations.

Knowledge Graph

Maintain a knowledge graph of program-analysis concepts, relations, and example challenges

Context-aware Follow-ups

Support context-aware follow-ups within a session to clarify, drill down, or chain questions.

Canonical Explanations

Ensure canonical explanations (consistent across sessions) by templating responses and pulling definitions from the graph.

Interactive Knowledge Graph

Implementation of a method to interact with the Knowledge Graph to insert, update, and query nodes and relationships.
